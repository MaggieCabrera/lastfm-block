Clean and simple recently played Last.FM Plugin for WordPress


== Description ==

Clean and simple recently played Last.FM Plugin for WordPress. This plugin is a fork of https://github.com/nicolasbettag/LastFM-Played-for-Wordpress to add block functionality so the plugin can be used on block themes.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory or install it via the admin panel
2. Activate the plugin through the 'Plugins' menu in WordPress

= Block =

3. Insert the block on a template or template part
4. Enter your Last.FM Username
5. Happy Scrobbling!

= Widget =

3. Add the widget to your sidebar
4. Enter your Last.FM Username
5. Happy Scrobbling!
